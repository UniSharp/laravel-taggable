## Laravel Data Carrier

### Introduction

this api can handle tag in independent tables or use only one table 

### Install Taggable 

composer.json:

    "require" : {
        "unisharp/laravel-taggable" : "dev-master"
    }, 
    "repositories": {
        "type": "git",
        "url": "https://github.com/UniSharp/laravel-taggable.git
    }

save it and then 

    composer update
